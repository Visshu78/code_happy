Asterisk Installation Checklist (Ubuntu)

Target OS:
- Ubuntu 20.04 / 22.04 (server)

System preparation:
- Update system packages
- Install build tools
- Install required libraries

Asterisk source:
- Download stable Asterisk LTS
- Compile with WebRTC support enabled

Key modules needed:
- SIP (PJSIP)
- RTP
- WebRTC (DTLS, SRTP)
- HTTP / WebSocket

Security basics:
- Run as non-root user
- Firewall rules
- Disable unused modules

Note:
Installation will be done later on a real server.
This document is only a reference.
